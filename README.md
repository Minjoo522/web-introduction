# web-introduction

## 요구사항

- [x] HTML 파일의 인코딩 방식은 'UTF-8'
- [x] 이미지 파일의 경로는 모두 상대 경로
- [x] `<head>` 안에는 `<meta charset="utf-8">`을 집어넣습니다
- [x] 본문은 `<div>`로 적절히 화면을 나누어야 합니다
- [x] 적절히 `<h#>`으로 섹션을 구분

### 이름 및 자기소개

- [x] 이름과 간단한 자기소개가 문단으로 구성

### 인생 영화 BEST5

- [x] 인생 영화 BEST5를 표로 정리
  - [x] 영화의 포스터와 바로가기 링크(네이버 영화)는 꼭

### 방명록

- [x] 방문자가 남길 수 있는 방명록
  - [x] 이름은 `<input>`으로 type은 'text'
  - [x] 방문자가 본인의 취향의 영화를 고를 수 있도록 체크박스 형태로 제공
- [ ] 등록 버튼과 취소 버튼
  - [x] 등록 버튼은 `<button>`을 사용하며 type은 'button'
  - [x] 취소 버튼은 `<button>`을 사용하며 type은 'reset'
- [x] 방명록에 등록한 사람 이름(OOO)과 일치하는 취향의 개수(N)를 조사하여 “OOO님, 저와 N개의 취향이 같으시네요!"라는 Alert 띄우기
